# Termux Launcher Site

Static product, setup, documentation, and showcase site for [Termux Launcher](https://github.com/PickleHik3/termux-launcher).

## Local preview

```sh
python -m http.server 8000
```

Open <http://127.0.0.1:8000>.

The production site is published with GitHub Pages at <https://picklehik3.github.io/termux-launcher-site/>.
